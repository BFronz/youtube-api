# YouTube Clone
 The app fetches searched YouTube videos.

### Installing

```
 
clone https://github.com/BFronz/youtube-api.git

npm install package.json

Run  npm start

```

## Built With
* React.js
* Material UI
* API data fetching 
* Node.js
* Axios
